## How to record `demo.gif`

1. Load the extension unpacked (`chrome://extensions` → Developer mode → Load unpacked).
2. Open a news site that has sports articles in its feed.
3. Start a screen recorder (e.g. ScreenToGif on Windows, or Chrome's built-in recorder).
4. Show the sequence:
   - the feed with sports articles visible,
   - open the ClearFeed popup, confirm Sport is enabled (or toggle it),
   - the sports articles disappearing from the feed,
   - the "articles hidden" counter going up.
5. Export as `demo.gif` (aim for < 5 MB, ~800px wide) and save it here.

Store screenshots (1280×800) can be captured the same way as still frames — see
`store/STORE_LISTING.md` for the recommended screenshot set.
